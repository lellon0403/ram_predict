# PC 부품 가격 예측 서비스 — 다음 발표 기획안

> 작성일: 2026-04-28  
> 참여자: 렐론, 이동욱

---

## 1. 핵심 방향 요약

**"RAM 언제 사지?" → "컴퓨터 언제 맞추지?"**

RAM 단일 예측에서 **PC 전체 부품 가격 예측 + 조립 추천 서비스**로 확장한다.  
PPT 발표 대신 **실제 동작하는 React 페이지**로 발표한다.

---

## 2. 확장할 부품 목록

| 부품 | 비고 |
|------|------|
| RAM | 기존 완성 (삼성 DDR5 16GB) |
| CPU | 신규 (예: Ryzen 9600X) |
| VGA (그래픽카드) | 신규 (예: RTX 5070) |
| 메인보드 | 신규 (예: MSI B650 WiFi) |
| SSD | 신규 |

> RAM 예측 모델 구조를 그대로 재사용하고 **CSV 데이터만 교체**하면 됨

---

## 3. 서비스 기능 목록

### 3-1. 개별 부품 예측
- 각 부품별로 RAM과 동일한 방식으로 가격 예측
- React 내에서 부품별 탭/페이지로 개별 확인 가능

### 3-2. 조립 PC 통합 가격 예측
- 5개 부품 가격 합산 → **"이 사양의 PC를 언제 사는 게 좋은가?"**
- 대표 사양 예시:
  ```
  CPU:  AMD Ryzen 9600X
  GPU:  RTX 5070
  RAM:  삼성 DDR5 16GB × 2
  보드: MSI B650 WiFi
  SSD:  (선정 예정)
  ```

### 3-3. 완본체 가격 비교
- 현재 판매 중인 완성형 PC와 조립 견적 비교
- "직접 조립 vs 완본체 구매" 어느 쪽이 이득인지 추천

### 3-4. 실시간 데이터 연동 (n8n)
- 쇼핑몰 API 또는 크롤링으로 실시간 최저가 수집
  - 후보: 아마존, 쿠팡, 롯데몰, 다나와
- 달러 기준 부품은 **실시간 환율 연동**
- n8n으로 외부 HTTPS 데이터 수집 자동화

### 3-5. 모델 개선
- 현재 (v1): 선형 회귀 → 직선
- 개선 (v2): 다항 회귀 → 곡선
- 오차율 8.53% 이하 목표

---

## 4. 발표 연출 계획

```
1. 기존 RAM 예측 (직선) 보여주기
        ↓
2. 곡선 모델로 개선된 결과 보여주기
        ↓
3. RAM이 슉 → 메인보드에 꽂히는 애니메이션
        ↓
4. CPU, GPU, SSD 순서로 조립되는 연출
        ↓
5. 완성된 PC 모양 등장
        ↓
6. "이 PC, 언제 맞추면 좋을까요?" → 예측 결과
```

- PPT 대신 **CSS 반응형 React 페이지**로 발표 진행
- 이동욱 노트북에서 실시간 동작

---

## 5. 기술 스택 변경사항

| 항목 | v1 (현재) | v2 (다음) |
|------|-----------|-----------|
| 모델 | 단일 변수 선형 회귀 | 다항 회귀 (곡선) |
| 데이터 | 수동 CSV | 실시간 API/크롤링 |
| 환율 | 없음 | n8n 실시간 환율 연동 |
| 부품 수 | 1개 (RAM) | 5개 |
| 발표 방식 | PPT | React 라이브 시연 |
| 외부 연동 | 없음 | 쇼핑몰 API + 환율 API |
